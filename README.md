## 🚗 Car Price Prediction using Machine Learning

## 📌 About the Project

This project was completed as part of my CodeAlpha Data Science Internship. The aim of this project is to predict car prices using machine learning by analyzing different car features such as engine size, horsepower, fuel type, car body, and other specifications.

The project covers the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), model building, and performance evaluation.

---

## 🎯 Project Objectives

- Explore the car price dataset.
- Clean and preprocess the data.
- Visualize important relationships between features.
- Train machine learning models to predict car prices.
- Compare model performance and select the best one.

---

## 📂 Dataset Information

The dataset contains information about 205 cars with features such as:

- Car Name
- Fuel Type
- Aspiration
- Car Body
- Drive Wheel
- Engine Size
- Horsepower
- Wheelbase
- Car Dimensions
- Price

## 📊 Exploratory Data Analysis

During EDA, the following visualizations were created:

- Distribution of Car Prices
- Correlation Heatmap
- Engine Size vs Price
- Horsepower vs Price
- Fuel Type Distribution
- Average Price by Car Body

These visualizations helped identify the features that have the greatest impact on car prices.

---

## 🤖 Machine Learning Models

Two regression models were implemented:

- Linear Regression
- Random Forest Regressor

### Model Performance

## 📈 Model Performance

| Machine Learning Model | R² Score | Performance |
| :--------------------- | :------: | :---------: |
| Linear Regression      | 0.8441   | Good |
| Random Forest Regressor| 0.9572   | Best ✅ |
The Random Forest model produced better predictions and achieved the highest accuracy.

---

## 💡 Key Findings

- Engine size and horsepower have a strong influence on car prices.
- Cars with larger engines generally cost more.
- Petrol cars are more common than diesel cars in this dataset.
- Random Forest performed better than Linear Regression for price prediction.
